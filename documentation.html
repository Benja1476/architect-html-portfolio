Documentation

Thank you for purchasing **Architect**. This documentation will help you set up and customize your new portfolio. Architect is built with Tailwind CSS, ensuring it is lightweight, fast, and easy to modify.

File Structure

Inside the main ZIP file, you will find:

/architect-template
  ├── index.html        (Main Portfolio File)
  ├── documentation.html (This guide)
  └── readme.txt        (Quick start info)

Customization

Changing Text & Content

Open index.html in any text editor (like VS Code or Notepad++) and search for the text you want to change. Most content is wrapped in standard HTML tags like <h1>, <p>, and <span>.

Dark/Light Mode

The theme uses a simple JavaScript function toggleTheme() located at the bottom of the file to switch between modes. You can set the default mode by changing the class on the <body> tag to either dark or light.

Deployment

To make your site live:

  1. Upload your files to a web server via FTP or use services like GitHub Pages / Netlify.
  2. Ensure index.html is in the root directory.
  3. Your site is ready!
